Das Problem mit Ajax ist, dass wenn wir eine Ajax-Anfrage senden und sie auf dem Server empfangen, wir auf dem Server keine Weiterleitung durchführen können.Deshalb die Antwort im JSON-Format weiterleiten und an den Frontend-Bereich senden, dann dort empfangen und die Weiterleitung durchführen.
